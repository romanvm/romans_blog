cerulean_skin.templatetags.cerulean_skin_tags.truncate_post(post)[source]¶

Filter

Truncate a Post text to a page separator. The separator is inserted by TinyMCE pagebreak plugin that is used to emulate “blog post cut” feature.

Only the 1st separator is taken into account, other separators, if any, are ignored.

Parameters:post – blog post
Returns:properly terminated truncated post
